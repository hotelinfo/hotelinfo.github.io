Комплекс для відпочинку Village Vacances Le Junka знаходиться в 800 метрах від пляжу. До ваших послуг відкритий басейн, дитячий ігровий майданчик, ресторан і безкоштовний Wi-fi у зонах загального користування. Гостям комплексу для відпочинку Village Vacances Le Junka пропонуються номери з видом на сад або город, телевізором з плоским екраном та окремою ванною кімнатою з душем і туалетом. Щодня для гостей сервірується континентальний сніданок. У ресторані подають страви традиційної французької кухні. Серед додаткових вигод на території комплексу для відпочинку - прокат велосипедів та ігровий зал. Працює команда аніматорів.
